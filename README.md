这是 HRP 商业网站的 1亿目标重构版。

- AiQ 从 3万人目标升级为 5万人目标，作为规模入口。
- Neural Cuff 从 2,200套/4家POC 升级为 3,000套/10家POC。
- 梦境设计从单一稀缺服务升级为四层矩阵：顶级版、进阶版、初探版、企业版。
- IP / 品牌共创从附属收入升级为主现金流引擎。
- AstroTimeCandle 从小产品升级为礼盒、联名、入口IP。
- 年度收入模型重构为约 ¥1.15亿。
